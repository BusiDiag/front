# BusiDiag

Frontend built for BusiDiag.

## Pages

### Businesses

This page is for managing all the businesses added to the app. Adding and deleting business is available.

### Results

This is where all the results are shown and managed. User will be able to select one of the business (if one was selected before in the Businesses section, the one selected will show up), and selecting one wanted will show all the results made beforehand.

## Stacks

- React
- Tailwind CSS
